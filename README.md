# Compilador Dislexia

Proyecto del curso de Compiladores.

## Descripción

Este proyecto implementa un compilador básico utilizando C++, donde procesa el texto ingresado y devuelve las palabras con posibles errores ortográficos para personas Disléxicas.

## Compilación

### Crear carpeta build

```bash
cd build
cmake ..
make
cd ..
./build/compilador
```
